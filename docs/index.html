What's the difference between the usage symbols?

  • <param>: This param (parameter) is mandatory for the command to function and cannot be omitted.
  • [param]: This param (parameter) is not mandatory for the command to function and can be omitted, but it provides extra auxiliary features.
  • (param): This param (parameter) is mandatory for the command to function in the aforementioned way and cannot be omitted unless you don't want this behavior.

What's the difference between the docs and the FAQ?

  • These docs are for using the bot itself, some tips/tricks, and common questions as a substitute for asking them in support.
  • The FAQ is only for common questions as a substitute for asking them in support.
