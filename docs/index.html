What's the difference between the usage symbols?

  • <param>: This param, or parameter, is mandatory for the command to function and cannot be omitted. Alias: Info Param/IP.
  • [param]: This param, or parameter, is not mandatory for the command to function and can be omitted, but it provides extra auxiliary features. Alias: Additional Param/AP.
  • (param): This param, or parameter, is mandatory for the command to function in the aforementioned way and cannot be omitted unless you don't want this behavior. Alias: Technical Additional Param/TAP.
